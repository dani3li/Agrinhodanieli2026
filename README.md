# 🌱 EcoFeira Local - Marketplace Sustentável

## Sobre o Projeto

**EcoFeira Local** é um site inovador de marketplace sustentável desenvolvido com o tema **"Agro Forte e Futuro Sustentável"**. O projeto conecta consumidores conscientes a pequenos produtores locais, promovendo o consumo responsável, reduzindo a pegada de carbono e apoiando a agricultura sustentável.

Este projeto foi criado para incentivar:
- ✅ **Consumo Consciente** - Escolher produtos de produtores locais e sustentáveis
- ✅ **Redução de Pegada de Carbono** - Diminuir impacto ambiental com compras locais
- ✅ **Apoio a Produtores Locais** - Fortalecer pequenas fazendas e empreendimentos familiares
- ✅ **Educação Ambiental** - Promover práticas agrícolas sustentáveis (alinhado ao Concurso Agrinho)

## 🚀 Como Usar

### **1. Instalação**
1. Clone ou baixe este repositório
2. Extraia os arquivos em uma pasta
3. Certifique-se de que as pastas `imagens/` estão no mesmo diretório

### **2. Abrir no Navegador**
- Simplesmente abra o arquivo `index.html` em qualquer navegador moderno
- Não requer servidor ou instalação adicional

### **3. Navegar no Site**
- Use o menu do header para navegar entre seções
- Clique em "Explore Agora" no hero para ir direto ao catálogo
- Use o scroll suave para navegar entre seções

## 💡 Recursos de Interatividade

### **Filtragem de Produtos**
```
1. Acesse a seção "Catálogo de Produtos"
2. Clique em um dos filtros: "Todos", "Orgânicos" ou "Sazonais"
3. Os produtos são filtrados instantaneamente
4. Clique novamente para mudar o filtro
```

### **Adicionar à Lista de Interesse**
```
1. Localize o produto desejado
2. Clique no botão "+ Interesse"
3. Você receberá uma notificação confirmando
4. O botão muda temporariamente para "✓ Adicionado!"
```

### **Saber Mais sobre Produtores**
```
1. Clique no botão "Saber Mais" de qualquer produtor
2. Um modal aparecerá com informações detalhadas
3. Clique em "Fechar" ou fora do modal para sair
```

### **Navegação Suave**
- Clique em qualquer link do menu
- O site faz scroll suave até a seção desejada
- A navegação é fluida e agradável

## 🔧 Personalizações

### **Mudar Cores**
Edite as variáveis CSS em `style.css`:
```css
:root {
    --primary-green: #2ECC71;
    --dark-green: #27AE60;
    /* ... outras cores ... */
}
```

### **Adicionar Novos Produtos**
1. Adicione uma nova `<div class="product-card">` na seção de catálogo
2. Defina o `data-category` (organicos ou sazonais)
3. Adicione a imagem de fundo
4. Preencha as informações do produto

### **Adicionar Novos Produtores**
1. Crie um novo `<div class="producer-card">` na seção "Quem Produz"
2. Adicione a imagem do produtor
3. Preencha com as informações desejadas
